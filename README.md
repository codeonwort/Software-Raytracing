# Goal

Implement CPU ray tracing

# Environment

* Language   : C++11
* IDE        : Visual Studio 2019 or newer
* Build tool : Cmake
* OS         : Windows 10 or newer

# Sample images

Most of OBJ models are downloaded from Morgan McGuire's [Computer Graphics Archive](https://casual-effects.com/data)
